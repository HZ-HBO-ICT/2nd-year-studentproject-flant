# Flant dev setup

Install postgres
https://www.postgresql.org/   
Restart your computer

**In the project directory :**  
**In the terminal**
**installs packages in the global folder**  
_Npm install_  

**Cd client** 
**installs packages in the client folder**  
_Npm install_  
**start the front end**  
_Npm start_  

**in another terminal**  
_Cd server_  
**installs packages in the server folder**  
_Npm install_  
**start the back end**  
_Npm run dev_
